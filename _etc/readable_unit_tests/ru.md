## –ß—Ç–æ —Ç–∞–∫–æ–µ BDD?

## BDD —Å—Ç–∏–ª—å

–ß—Ç–æ —è –ø–æ–¥—Ä–∞–∑—É–º–µ–≤–∞—é –ø–æ–¥ –±–æ–ª–µ–µ **—á–∏—Ç–∞–±–µ–ª—å–Ω—ã–º–∏** —Ç–µ—Å—Ç–∞–º–∏?

–≠—Ç–æ **BDD —Å—Ç–∏–ª—å** –Ω–∞–ø–∏—Å–∞–Ω–∏—è —Å—Ü–µ–Ω–∞—Ä–∏–µ–≤ (–∏–ª–∏ —Ç–µ—Å—Ç–æ–≤).

```text

  And holding value should be $2000
```

BDD —Å—Ü–µ–Ω–∞—Ä–∏–∏ (—Ç–µ—Å—Ç—ã) –ø–∏—à—É—Ç—Å—è –ø–æ —Å–ª–µ–¥—É—é—â–µ–º—É —à–∞–±–ª–æ–Ω—É **GIVEN-WHEN-THEN**:

- **WHEN**  - –¥–µ–π—Å—Ç–≤–∏–µ, –∫–æ—Ç–æ—Ä–æ–µ —Ç–µ—Å—Ç–∏—Ä—É–µ–º, —Ç.–µ. —Ü–µ–Ω–∞ –º–æ–Ω–µ—Ç–∫–∏ –≤—ã—Ä–æ—Å–ª–∞ –≤ –¥–≤–∞ —Ä–∞–∑–∞ –∏ –∫–∞–ª—å–∫—É–ª—è—Ç–æ—Ä –Ω–∞–º –¥–æ–ª–∂–µ–Ω –ø–æ—Å—á–∏—Ç–∞—Ç—å –Ω–æ–≤—ã–µ –∑–Ω–∞—á–µ–Ω–∏—è;
- **THEN** - –≤—ã—Ö–æ–¥–Ω—ã–µ –ø–∞—Ä–∞–º–µ—Ç—Ä—ã –∏–ª–∏ –∫–æ–Ω–µ—á–Ω–æ–µ —Å–æ—Å—Ç–æ—è–Ω–∏–µ, —Ç.–µ. –Ω–æ–≤—ã–µ –∑–Ω–∞—á–µ–Ω–∏—è, —É –Ω–∞—Å –ø—Ä–æ—Ñ–∏—Ç, —É–±—ã—Ç–æ–∫ –∏–ª–∏ –Ω–∏—á–µ–≥–æ –Ω–µ –∏–∑–º–µ–Ω–∏–ª–æ—Å—å.

## –¢—Ä–∏ —á–∞—Å—Ç–∏ —é–Ω–∏—Ç-—Ç–µ—Å—Ç–æ–≤

–î–∞–ª–µ–µ, –Ω–∞–º –Ω—É–∂–Ω–æ  –≤—ã–¥–µ–ª–∏—Ç—å —Ç—Ä–∏ —á–∞—Å—Ç–∏, –∫–æ—Ç–æ—Ä—ã–µ —Å–≤—è–∑–∞–Ω—ã —Å —é–Ω–∏—Ç-—Ç–µ—Å—Ç–∞–º–∏. –≠—Ç–æ:

- –∫–æ–¥ —Å–∞–º–∏—Ö —Ç–µ—Å—Ç–æ–≤;
- –æ—Ç—á–µ—Ç –æ–± —É—Å–ø–µ—Ö–µ –∏–ª–∏ –æ –ø—Ä–æ–≤–∞–ª–µ;
- —Å–æ–æ–±—â–µ–Ω–∏–µ –æ–± –æ—à–∏–±–∫–µ –≤ —Å–ª—É—á–∞–µ –ø—Ä–æ–≤–∞–ª–µ–Ω–Ω–æ–≥–æ —Ç–µ—Å—Ç–∞.

–ö–∞–∂–¥–∞—è –∏–∑ —ç—Ç–∏—Ö —á–∞—Å—Ç–µ–π –æ—á–µ–Ω—å –≤–∞–∂–Ω–∞:

- **–û—Ç—á–µ—Ç** - –º–æ–∂–µ—Ç —è–≤–ª—è—Ç—å—Å—è "–∂–∏–≤–æ–π –¥–æ–∫—É–º–µ–Ω—Ç–∞—Ü–∏–µ–π", –∫–æ—Ç–æ—Ä–∞—è –æ–ø–∏—Å—ã–≤–∞–µ—Ç –∫–∞–∫ –ø—Ä–æ–≥—Ä–∞–º–º–∞ —Ä–∞–±–æ—Ç–∞–µ—Ç, –∏ –∫–æ—Ç–æ—Ä—É—é —Å–º–æ–∂–µ—Ç –ø—Ä–æ—á–µ—Å—Ç—å –ª—é–±–æ–π —á–ª–µ–Ω –∫–æ–º–∞–Ω–¥—ã –∏ –¥–∞–∂–µ –Ω–µ —Ç–µ—Ö–Ω–∏—á–µ—Å–∫–∏–π —Å–ø–µ—Ü–∏–∞–ª–∏—Å—Ç.
- –ï—Å–ª–∏ **—Å–æ–æ–±—â–µ–Ω–∏–µ –æ–± –æ—à–∏–±–∫–µ** —è—Å–Ω–æ —Ä–∞–∑—ä—è—Å–Ω—è–µ—Ç –ø—Ä–∏—á–∏–Ω—É –æ—à–∏–±–∫–∏, –ø–æ–Ω—è—Ç—å –∫–æ–Ω—Ç–µ–∫—Å—Ç, —Ç–æ –∏—Å–ø—Ä–∞–≤–∏—Ç—å —Ç–µ—Å—Ç –±—É–¥–µ—Ç –±—ã—Å—Ç—Ä–µ–µ.

–ö–∞–∂–¥—É—é –∏–∑ —ç—Ç–∏—Ö —á–∞—Å—Ç–µ–π —Å–µ–π—á–∞—Å –º—ã –±—É–¥–µ–º —É–ª—É—á—à–∞—Ç—å.

## PNL Calculator

–ù–∞ –ø—Ä–∏–º–µ—Ä–µ `PNL –∫–∞–ª—å–∫—É–ª—è—Ç–æ—Ä–∞` –º—ã –±—É–¥–µ–º –æ–ø–∏—Å—ã–≤–∞—Ç—å –µ–≥–æ –ø–æ–≤–µ–¥–µ–Ω–∏–µ –≤ —é–Ω–∏—Ç-—Ç–µ—Å—Ç–∞—Ö.

–ü—Ä–µ–¥—Å—Ç–∞–≤–∏–º, —á—Ç–æ –Ω–∞–º –Ω—É–∂–Ω–æ –Ω–∞–ø–∏—Å–∞—Ç—å —Ç–µ—Å—Ç—ã –Ω–∞ —Ç—Ä–∏ —Å–ª—É—á–∞—è:

- –∫–æ–≥–¥–∞ —Ü–µ–Ω–∞ –º–æ–Ω–µ—Ç–∫–∏ –≤—ã—Ä–æ—Å–ª–∞ - –ø—Ä–æ—Ñ–∏—Ç (profit);
- –∫–æ–≥–¥–∞ —Ü–µ–Ω–∞ –º–æ–Ω–µ—Ç–∫–∏ —Å–Ω–∏–∑–∏–ª–∞—Å—å - —É–±—ã—Ç–æ–∫ (loss);
- –∫–æ–≥–¥–∞ —Ü–µ–Ω–∞ –º–æ–Ω–µ—Ç–∫–∏ –Ω–µ –∏–∑–º–µ–Ω–∏–ª–∞—Å—å.

BDD —Å—Ü–µ–Ω–∞—Ä–∏–π –¥–ª—è —Å–ª—É—á–∞—è –ø—Ä–æ—Ñ–∏—Ç–∞:

```text
Given 1000 coins with a total investment of $1000
When coin price raise up to $2
Then PNL % should be 100%
  And PNL value should be $1000
  And holding value should be $2000
```

–Æ–Ω–∏—Ç-—Ç–µ—Å—Ç –º–æ–≥ –±—ã –≤—ã–≥–ª—è–¥–µ—Ç—å —Å–ª–µ–¥—É—é—â–∏–º –æ–±—Ä–∞–∑–æ–º:

```dart
group('Given 1000 coins with a total investment of \$1000 ', () {
  const coinAmount = 1000.0;
  const totalInvestment = 1000.0;

  group('\nWhen coin price raise up to \$2', () {
    late PnlCalculatorResult pnl;
    setUp(() {
      pnl = PnlCalculator.calculate(
        coinAmount: coinAmount,
        totalInvestments: totalInvestment,
        currentPrice: 2,
      );
    });
    test('\nThen PNL % should be 100%', () {
      expect(pnl.percent, 100);
    });

    test('\n    And PNL value should be \$1000', () {
      expect(pnl.value, 1000);
    });

    test('\n    And holding value should be \$2000', () {
      expect(pnl.holdingValue, 2000);
    });
  });
});
```

–ò –Ω–∞ –≤—ã—Ö–æ–¥–µ –º—ã –±—ã –ø–æ–ª—É—á–∏–ª–∏, —á—Ç–æ-—Ç–æ –≤—Ä–æ–¥–µ —Ç–∞–∫–æ–≥–æ:

```bash
‚úì Given 1000 coins with a total investment of $1000  
When coin price raise up to $2 
Then PNL % should be 100%
‚úì Given 1000 coins with a total investment of $1000  
When coin price raise up to $2 
    And PNL value should be $1000
‚úì Given 1000 coins with a total investment of $1000  
When coin price raise up to $2 
    And holding value should be $2000
```

–ö–∞–∫–∏–µ –µ—Å—Ç—å –ø—Ä–æ–±–ª–µ–º—ã c —ç—Ç–∏–º —Ç–µ—Å—Ç–æ–º –∏ –æ—Ç—á–µ—Ç–æ–º?

- –ö–æ–¥ —Ç–µ—Å—Ç–∞ —Ç—è–∂–µ–ª–æ —á–∏—Ç–∞—Ç—å. –¢—É—Ç –µ—Å—Ç—å –∏ `group`, `setUp` –∏ `test` —Å–ª–æ–≤–∞, –∫–æ—Ç–æ—Ä—ã–µ —Ç–µ—Ö–Ω–∏—á–µ—Å–∫–∏ —Å–ø–µ—Ü–∏–∞–ª–∏–∑–∏—Ä–æ–≤–∞–Ω—ã –∏ –º–µ—à–∞—é—Ç –ø—Ä–æ—á–∏—Ç–∞—Ç—å –æ —á–µ–º —Ç–µ—Å—Ç. –ê BDD –ø—Ä–µ–¥–ø–æ–ª–∞–≥–∞–µ—Ç –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ –±–æ–ª–µ–µ –ø—Ä–æ—Å—Ç—ã—Ö —Å–ª–æ–≤ –∏ —Ñ–æ—Ä–º—É–ª–∏—Ä–æ–≤–æ–∫, —á—Ç–æ –ª—é–±–æ–π —É—á–∞—Å—Ç–Ω–∏–∫ –∫–æ–º–∞–Ω–¥—ã, –¥–∞–∂–µ –¥–∞–ª–µ–∫–∏–π –æ—Ç –ø—Ä–æ–≥—Ä–∞–º–º–∏—Ä–æ–≤–∞–Ω–∏—è —á–µ–ª–æ–≤–µ–∫, —Å–º–æ–∂–µ—Ç –ø—Ä–æ—á–∏—Ç–∞—Ç—å —Ç–µ—Å—Ç –∏ –ø–æ–Ω—è—Ç—å –æ —á–µ–º –æ–Ω.
- –ïc–ª–∏ –ø–æ—Å–º–æ—Ç—Ä–µ—Ç—å –Ω–∞ –æ—Ç—á–µ—Ç, —Ç–æ –∑–∞–º–µ—Ç–∏–º –¥—É–±–ª–∏—Ä–æ–≤–∞–Ω–∏–µ –±–ª–æ–∫–æ–≤ `GIVEN` –∏ `WHEN`, –∏ —Å–µ–π—á–∞—Å –Ω–µ –ø—Ä–æ—Å—Ç–æ –¥–æ–±–∏—Ç—å—Å—è —Å–ª–∏—è–Ω–∏—è –±–ª–æ–∫–∞ `THEN` –∏ `AND`, —á—Ç–æ–±—ã –æ—Ç—á–µ—Ç –≤—ã–≥–ª—è–¥–µ–ª —Ä–æ–≤–Ω–æ —Ç–∞–∫, –∫–∞–∫ —Å–∞–º —Å—Ü–µ–Ω–∞—Ä–∏–π. –ò `\n` —Å —Ç–∞–±—É–ª—è—Ü–∏–µ–π –≤ –æ–ø–∏—Å–∞–Ω–∏–∏ —Ç–µ—Å—Ç–∞ —Ç–∞–∫ –∂–µ –ø–æ—Ä—Ç—è—Ç –∫–æ–¥ —Å–∞–º–æ–≥–æ —Ç–µ—Å—Ç–∞.

–ò —Ç—É—Ç –∫ –Ω–∞–º –Ω–∞ –ø–æ–º–æ—â—å –ø—Ä–∏—Ö–æ–¥–∏—Ç –ø–µ—Ä–≤—ã–π Dart-–ø–∞–∫–µ—Ç [given_when_then_unit_test](https://pub.dev/packages/given_when_then_unit_test). –° –ø–æ–º–æ—â—å—é –∫–æ—Ç–æ—Ä–æ–≥–æ, –Ω–∞—à–∏ —Ç–µ—Å—Ç—ã –±—É–¥—É—Ç –∑–∞–ø–∏—Å–∞–Ω—ã —Å–ª–µ–¥—É—é—â–∏–º –æ–±—Ä–∞–∑–æ–º:

```dart

  const totalInvestment = 1000.0;

        coinAmount: coinAmount,
        totalInvestments: totalInvestment,
        currentPrice: 2,
      );
```
